# Algoritmos-Fundamentais-em-Faccat-JS
exercícios de lógica básica em JavaScript. Inclui soluções para cálculos de área/percentual, par/ímpar, estruturas condicionais (if/else if), operadores lógicos (&amp;&amp;, ||), e aninhamento para classificação de dados e regras de elegibilidade. Foco em estruturas de decisão.
